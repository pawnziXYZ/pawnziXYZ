## Pawnzi

Collateral-backed SOL liquidity protocol built on Solana.

Pawnzi enables users to lock supported crypto assets reminding collateral and access short-term SOL liquidity without selling their assets. Loan terms are fixed, transparent, and settled automatically.

### What Pawnzi Offers
- Short-term SOL liquidity backed by on-chain collateral
- Clear loan terms with predefined duration and fees
- Automatic settlement at loan maturity
- Asset-backed borrowing without forced liquidation during the loan period

### Status
Pawnzi is live and actively maintained on Solana mainnet.

Core protocol components are under continuous review to ensure stability, security, and reliable execution.

### Open Source Status
The Pawnzi codebase is not open-sourced at this stage.

This approach allows the team to complete security reviews, internal audits, and infrastructure hardening before any public code disclosure.
